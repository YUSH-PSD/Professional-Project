This repository contains a simple hotel landing page demo built with **React + Vite + Tailwind CSS**. It demonstrates a full-screen hero image, a floating navigation bar, and a booking panel that you can use as a starting point for hospitality landing pages.

- Entry: `src/main.jsx`
- Main component: `src/App.jsx`

Run locally with:

```bash
npm install
npm run dev
```
